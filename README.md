# comments

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development (serves the vue app)
```
yarn run serve
```

### Run REST API server
```
yarn run dev-api
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your end-to-end tests
```
yarn run test:e2e
```

### Run your unit tests
```
yarn run test:unit
```
